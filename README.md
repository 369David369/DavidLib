***Maven***
Mosaik is on the Maven central.
```xml
<dependency>
  <groupId>io.github.splotycode.mosaik</groupId>
  <artifactId>Mosaik-All</artifactId>
  <version>2.2.9</version>
</dependency>
```
Note: GameEngine and SpigotLib are not part of Mosaik-All
